Using Vuforia Multi-target in Unity

Sep 13, 2017

今天我成功的使用Vuforia里面的Multi target识别了多个子marker。主要方法有两种：

  1. 在Vuforia develop portal里面生成Multi-target。新建target的时候可以有两种选择，分别是立方体和圆柱体。拿立方体而言，上传的时候要自定义长宽高。每个面的target都要精确的按照比例上传。所以使用这种方法限制比较多。

  2. 另一种方法就是修改XML文件，手动添加Multi target。这种方法更加方便与灵活。比如这样

    第二种方法可以使用任意形状。不会被限制在立方体或者圆柱体这两种形状中。我们可以添加任意多个childtarget（最多试验过8个），并且可以将这些childtarget按照我们想要的位置和方向排列（其实在第一个方法中也可以手动修改XML文件改变本来长方体的形状）。有一个比较便捷的方式来修改XML文件，就是先将childtarget在Unity里面拖动到想要的位置，然后把它的坐标和方向copy到XML中就可以了。注意，y和z在Unity与XML文件中对调了。

    另外需要注意的是，第二种方法里的每个childtarget必须已经在Vuforia中单独生成。

Unity - 我所遇到的问题

Sep 12, 2017

这篇博文将持续记录我在试用Unity的过程中遇到的，已经解决或者尚未解决的问题。

  • 2017.9.12
    今天我想在Unity中给Vuforia添加Multi target，目的是通过HoloLens识别多个markers来给物体定位。为此我用Photoshop将一个大marker拆解成了多个小marker，然后上传至Vuforia生成database。结果在将新的database导入之后发现任何一个marker都不能被HoloLens识别。

    经过研究后发现，发现如果Unity中如果导入了多个Vuforia的database，需要在Vuforia configuration里面勾选需要用到的database，这样HoloLens就可以识别了。
